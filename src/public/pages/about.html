About QA Ticks and Crosses

We used to have the ability in WebEx Training for learners to send us ticks, crosses and other emojis. In WebEx Meetings no such feature exists. This web app aims to fill that gap.

Support

This application has been developed by Andrew Burgess, a ex-tutor at QA, in his spare time. It is not supported by Andrew, QA support desk, or any other function within QA. If something goes wrong, please post a message in the tutor chat on MS Teams and hopefully someone will be able to help. Suggestions for improvements are always welcome.

If you want to access the original application code, the source code is available on GitHub.

Pages

  • home (the home page where you makeup a new room for your class)
  • about (this page)
  • settings (for testing multiple learners in multiple tabs)
  • browser-check (test all the elements in a browser)

Changes

Historical changes

  • Ability to highlight multiple learners at once
  • Smiley restored to bottom row, and new 'double-smiley' added to top row
  • Counter added for how many learners are actively connected to the room
  • Added option to only show learners with a status flag, hiding those with blank status
  • Changed 'kick out' feature so it clears all learners - all learners actually connected will reapper shortly afterwards
  • Improved distinction between status icon colours
  • Better auto-recovery of status lists in the event of a temporary server failure
  • Faster update to the tutor's sceeen when a learner changes their status
  • Faster update to the learner's sceeen when tutor clears the status
  • Bigger buttons to click / tap, so less chance of hitting wrong one
  • More reliable beep when a learner raises their hand
  • Automatic removal of names of learners who have left the app (5 minutes grace period applies)

Recent changes

  • Room names can only consist of letters and numbers (no special characters)
  • Added poll feature to show counts of learner status
  • Added 5th letter option for learners to select - so now A, B, C, D, and E
  • Learner name restricted to letters and numbers, and max length of 20
  • Learners can now enter a free text answer which shows up on the tutor page
